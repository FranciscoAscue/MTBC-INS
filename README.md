#README

Este repositorio tiene scripts relacionados a el ensamblaje de genomas
Para que puedan ser ejecutados se deben instalar los siguientes progrmas:

FastQC
Trimmomatic
jellyfish
sratools(NCBI)
Entrez(NCBI)
bowtie2
samtools
SPAdes

Para descargar usar el comando

```git clone ```
```cd/scriptsgenome```
```chmod 777 *.bash```
